Simply Add PDF to your pages or post via shortcode or via WP Bakery Visual composer Element in a Flip Book Style.
== Description ==

To use the plugin you can simply add a short code with the below structure to any place of your website (Pages or Posts). Or you can simply use the WP Bakery Visual Composer Element included. Short Code Example
[pfbk_pdf_flipbook src="http//www.domain.com/file.pdf" width="700" height="500" pages="on" animation="on"]

*Features:*
WP Bakery Visual Composer Element (Included): This plugin will automatically add a new element to WP Bakery Visual Composer if you have Visual Composer Installed and activated.
Shortcode Options: This plugin will give you a set of options to enable/disable within the shortcode or from the Visual Composer Element.

*Shortcode Options:*
(src) is the url of the PDF file (REQUIRED), all the other attributes are optional
(width) is the width of the PDF file to be displayed
(height) is the Height of the PDF file to be diplayed
(pages) attribute (on) for showing the PDF file pages number, (off) for not showing it.
(animation) attribute (on) for showing the PDF flip pages , (off) for not showing it.

You can simply remove the attributes to turn it off.

== Installation ==

Simply Install the plugin and activate it, then got the PDF Flip Book page from the left menu to copy the shortcode, then past it to the required page or post.

e.g.

1. Upload the plugin files to the /wp-content/plugins/PDF-Flip-Book directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the PDF Flip Book screen from the left menu to copy the shortcode.
1. Paste the shortcode to the desired page or post.
1. Change the src attribute with the PDF file URL.


== Frequently Asked Questions ==

= What is the required attribute of the shortcode? =

Only the src attribute is required, which is the URL of the PDF file.

= What will happen if I added the shortcode with the src only? =

All the other options will be set to the default as long as you didn't define it in the shortcode.
